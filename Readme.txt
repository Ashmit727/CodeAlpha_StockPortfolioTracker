# 📈 Stock Portfolio Tracker

## 📌 Project Description

Stock Portfolio Tracker is a simple Python application developed as part of the CodeAlpha Python Programming Internship.

The program allows users to enter stock names and quantities, calculates the investment value for each stock, and displays the total portfolio value. It also saves the portfolio summary to a text file.

---

## ✨ Features

* Track multiple stocks
* Calculate investment value automatically
* Supports Indian and Global MNC stocks
* Displays complete portfolio summary
* Saves results to a text file
* User-friendly console interface

---

## 🛠️ Technologies Used

* Python 3
* Dictionary
* Loops
* Conditional Statements
* File Handling

3. Open Terminal.
4. Run the program:
